# Dataset Description

This project uses a telecom customer dataset to predict:
1. Next month data usage (Regression)
2. Customer churn (Classification)

## Features
- Customer demographics
- Subscription details
- Usage patterns
- Billing information

## Target Variables
- **Next_Month_Data_Usage**: Continuous numeric value
- **Churn status**: Binary (0 = No, 1 = Yes)

## Notes
- Original dataset is not uploaded due to privacy concerns
- A small anonymized sample is provided in `data/`
