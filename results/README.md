# Results

This folder contains outputs generated from model training and evaluation.

## Regression Results
- `metrics.txt` – MAE, MSE, RMSE, R² scores
- `predicted_vs_actual.png` – Linear Regression performance

## Classification Results
- `accuracy.txt` – Model accuracy
- `confusion_matrix.png` – Confusion matrix visualization
- `decision_boundary.png` – SVM decision boundary (2D projection)
